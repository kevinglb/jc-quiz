
--------------------------------------------------
记录openid接口：

url：api/index.php

请求方式：POST

请求参数：

action=add_user
openid
utm_source
utm_medium

返回值：json

result
= 0 失败
= 1 成功
= 100 不在活动期内

--------------------------------------------------
记录问题答案接口：

url：api/index.php

请求方式：POST

请求参数：

action=add_question
question_no：1-8
answer：格式：单选（1-6），多选（1,4）
answer_all：当提交第8题时提交是提交该值，格式：1;3;4;3;5;2,4;3;6

返回值：json

result
= 0 失败
= 1 成功
= 100 不在活动期内
